# Ucab

A MERN stack cab booking platform foundation.

## Project Structure
- `client/`: React frontend application (Vite)
- `server/`: Node.js backend API (Express + MongoDB)

## Setup Instructions

### Backend (server)
1. `cd server`
2. `npm install`
3. Copy `.env.example` to `.env` and fill in your variables.
4. `npm start` (or `npm run dev` if you configure nodemon later).

### Frontend (client)
1. `cd client`
2. `npm install`
3. Copy `.env.example` to `.env` and set `VITE_API_URL`.
4. `npm run dev`
